## 龙虾记忆
### 分层记忆
- **`MEMORY.md`** — 长期记忆。持久事实、偏好和决策。会在每个私信会话开始时加载。
- **`memory/YYYY-MM-DD.md`** — 每日笔记。持续上下文和观察记录。今天和昨天的笔记会自动加载。
- **`DREAMS.md`**（可选）— Dream Diary 和 Dreaming 扫描摘要，供人工审阅，包括有依据的历史回填条目。
### 最佳实践
- **日常流程**：开启 Memory Flush 自动写每日日志，日常对话不需要手动关心记忆落地。
- **整理频率**：手动操作的话每 3-7 天执行一次 `/dream`；配置自动的话每周 1 次凌晨执行即可。
- **质量把控**：每月手动检查一次 `MEMORY.md`，修正自动提炼的错误或冗余内容，保证长期记忆的准确性。
- **多项目场景**：每个工作区单独配置心跳规则，避免跨项目记忆互相干扰。
